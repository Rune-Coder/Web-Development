Bulls & Cows






Instructions

This a code breaking game where I will think of a four digit
number and you will have to guess that number. You will get a
maximum of ten chances to make the correct guess. There are
three columns in the game board, namely the bulls, the cows and
lastly for the guess. The bulls and cows are the hints to help
you guess correctly. For each number entered, the bulls will
show the number of digits that is correctly entered and in
the correct position. On the other hand, the cows will show
the number of digits that is correctly entered but in the
wrong position. So you will win only if you get a value of
4 and 0 in the bulls and cows columns respectively.

How to play

Enter your number using the virtual keyboard given.
The digits will get automatically placed in the desired boxes
As soon as you enter four digits you will be able to see
the bulls and cows value for that number. There are no "del"
or "clear" option in the keyboard. This is deliberately done
to increase the difficulty of the game. So be careful while
entering the digits.
